# CatGallery 🐱

**CatGallery** es una aplicación web de alto rendimiento diseñada para ofrecer una experiencia de usuario fluida y moderna. Este proyecto se centra en la **arquitectura limpia**, el manejo eficiente de datos asíncronos y la optimización de la interfaz de usuario.

---

## 🚀 Características Principales

### 1. Sistema de Carga Híbrido (Smart Scroll)
Implementación de un sistema de carga que equilibra la automatización con el control del usuario:
- **Scroll Infinito:** Utiliza la API `IntersectionObserver` para cargar contenido de forma fluida al detectar el final de la página.
- **Control Manual:** Botón de "Ver más" que actúa como disparador y centinela visual para garantizar una navegación consciente.

### 2. Gestión de Favoritos con Persistencia
- **Almacenamiento Local:** Integración con `LocalStorage` para que la selección de favoritos del usuario persista entre sesiones.
- **Sincronización del DOM:** Actualización inmediata de la interfaz al marcar o desmarcar elementos, asegurando coherencia visual sin recargas.

### 3. Navegación Single Page (SPA)
- Arquitectura de visualización dual que permite alternar instantáneamente entre la "Galería Global" y "Mis Favoritos" optimizando el rendimiento.

### 4. UI/UX y Rendimiento
- **Estados de Carga:** Feedback visual constante mediante loaders dinámicos.
- **Manejo de Errores:** Gestión de fallos de red mediante bloques de control profesionales.
- **Estética Moderna:** Uso de gradientes, animaciones y efectos de diseño contemporáneo.

---

## 🛠️ Stack Tecnológico

- **JavaScript (ES6+):** Lógica modular y programación asíncrona.
- **Vite:** Herramienta de construcción y entorno de desarrollo.
- **CSS3:** Flexbox, Grid y Animaciones avanzadas.
- **The Cat API:** Integración de servicios RESTful.
